Principais formatações

Negrito/Destaque

Temos o negrito não semântico utilizando a marcação <b/> que preza somente
pela forma.

Também temos o negrito semântico que da força para palavra e relevância utilizando
a marcação <strong/>.

Italico\Enfâse

Temos o italico não semântico que visa somente a forma, termo em italico

Também temos o italico semântico que visa dar destaque ao termo, termo em italico semântico.

Texto marcado

Podemos criar também um texto marcado usando a tag <mark/>
termo marcado.

Texto pequeno

Texto que irá ser usado para linhas de contrato, exemplo termo pequeno

Texto excluido

Podemos ler, mas esse texto marcado significa que foi excluido e que já não tem mais coesão, termo excluido.

Texto incluido

Devemos ler e esse texto foi incluido e precisamos dar enfâse, pois e importante termo adicionado.

Texto sobrescrito

Podemos usar para digitar coisas como x2 + ax + bx + c

Texto subescrito

Podemos usar para digitar coisas como h20
